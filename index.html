PostgreSQL (Open-Source Relational Database Management System (RDBMS))

PostgreSQL is a powerful, open-source relational database management system known for its reliability, extensibility, and compliance with SQL standards. It supports ACID transactions, advanced data types, and high availability, making it ideal for scalable applications.

+

MySQL (Relational Database Management System)

It is an open-source relational database management system (RDBMS) developed by MySQL AB and currently maintained by Oracle Corporation. MySQL is known for its speed, reliability, and ease of use, making it a popular choice for web applications, e-commerce platforms, and enterprise systems.

+

Firebase (Application Development Platform)

It is an application development platform created by Google that provides a variety of cloud services to simplify the creation and management of web and mobile applications. Firebase offers authentication, real-time databases, storage, hosting, cloud functions, and analytics, allowing developers to focus on user experience without worrying about server infrastructure.

+

MongoDB (Document-Oriented NoSQL Database)

It is an open-source NoSQL database management system designed to store data in JSON-like document format. MongoDB is highly scalable and flexible, making it ideal for applications that handle large volumes of data and require a dynamic structure. It offers features such as replication, sharding, and advanced queries, making it a popular choice for web development, Big Data, and artificial intelligence.

UML (Unified Modeling Language)

It is a visual modeling language used in software engineering to represent and design complex systems. UML allows for the creation of diagrams that show the structure, behavior, and relationships of elements in a system.
