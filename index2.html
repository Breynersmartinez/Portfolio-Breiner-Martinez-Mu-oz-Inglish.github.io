Habilidades tecnológicas

+

Java

Es un lenguaje de programación orientado a objetos ampliamente utilizado para desarrollar aplicaciones en diversas plataformas. Es conocido por su portabilidad, lo que significa que el código puede ejecutarse en diferentes sistemas operativos sin modificación, gracias al uso de la máquina virtual Java (JVM).

+

Spring Boot

Es un framework basado en Java que facilita la creación de aplicaciones web y microservicios. Simplifica la configuración y el desarrollo al proporcionar herramientas preconfiguradas y una arquitectura modular que reduce la cantidad de código necesario para iniciar proyectos.

+

GitHub

Es una plataforma de alojamiento y gestión de código fuente que utiliza Git. Permite a los desarrolladores colaborar en proyectos de software, controlar versiones, revisar cambios y gestionar ramas de desarrollo, todo desde una interfaz basada en la web.

+

Git

Es un sistema de control de versiones distribuido que rastrea los cambios en archivos y permite a los desarrolladores trabajar en colaboración. Git permite gestionar el historial del proyecto, crear ramas para probar nuevas ideas y fusionar cambios de forma eficiente.

+

SQL (Structured Query Language)

Es un lenguaje de programación utilizado para gestionar y manipular bases de datos relacionales. Con SQL, los usuarios pueden realizar operaciones como insertar, actualizar, eliminar y consultar datos almacenados en tablas.

+

UML (Unified Modeling Language)

Es un lenguaje de modelado visual utilizado en ingeniería de software para representar y diseñar sistemas complejos. UML permite crear diagramas que muestran la estructura, el comportamiento y las relaciones de los elementos en un sistema.
